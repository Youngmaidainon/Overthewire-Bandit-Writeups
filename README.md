# 🛡️ OverTheWire Bandit Writeups

คลังเก็บรวบรวมวิธีทำ (Writeups) บันทึกคำสั่ง และสรุปสิ่งสำคัญที่ได้เรียนรู้จากการตะลุยโจทย์ **OverTheWire Bandit** ซึ่งเป็น Wargame ยอดนิยมสำหรับฝึกฝนทักษะ Linux Command Line และแนวคิดระบบความปลอดภัยเบื้องต้น

---

## 📂 โครงสร้างโฟลเดอร์ (Repository Structure)

เนื้อหาถูกแบ่งออกเป็นกลุ่มเลเวลตามหัวข้อและระดับความยาก เพื่อความง่ายต่อการค้นหาและทบทวนดังนี้

**`level0-5/` — พื้นฐานระบบและการจัดการไฟล์**
* เรียนรู้การเชื่อมต่อระบบระยะไกลด้วย SSH
* ฝึกฝนการจัดการไฟล์และการค้นหาข้อมูลเบื้องต้นในระบบ Linux


**`level6-10/` — การกรองข้อมูลและการสืบค้นขั้นสูง**
* เน้นการกรองข้อมูลจำเพาะด้วยคำสั่งอย่าง `grep` และ `find`
* การวิเคราะห์โครงสร้างไฟล์ระบบและการดึงข้อความ (Strings) ออกจากไฟล์ Binary


**`level11-15/` — การถอดรหัสและการจัดการไฟล์บีบอัด**
* การแปลงและถอดรหัสข้อความรูปแบบต่างๆ (เช่น Rot13, Base64)
* เทคนิคการแตกไฟล์บีบอัด (Archive) ที่ถูกบีบซ้อนกันหลายเลเยอร์


**`level16-20/` — ความปลอดภัยและการเชื่อมต่อเครือข่าย**
* เรียนรู้การสแกนพอร์ตเครือข่าย (Network Ports) เพื่อหาช่องทางที่เปิดอยู่
* การเชื่อมต่อและสื่อสารข้อมูลอย่างปลอดภัยผ่านโปรโตคอล SSL/TLS


**`level21-25/` — ระบบงานอัตโนมัติและการเขียนสคริปต์**
* วิเคราะห์และตรวจสอบการทำงานของระบบงานตั้งเวลาอัตโนมัติ (Cronjobs)
* การเขียน Bash Script เบื้องต้นเพื่อประมวลผลข้อมูลและแก้โจทย์แบบอัตโนมัติ


**`level26-33/` — การแหกคุกระบบและการเจาะประวัติ Git**
* เทคนิคการหลุดออกจากเชลล์ที่ถูกจำกัดสิทธิ์ (Restricted Shell Escape)
* การสืบค้นและกู้คืนข้อมูลความลับจากประวัติการบันทึกซอฟต์แวร์ (Git History)

## 💡 สิ่งที่ได้เรียนรู้โดยรวม (Overall Takeaways)

> **"จาก Command Line พื้นฐาน สู่ความเข้าใจในโครงสร้างระบบความปลอดภัย"**

* **การใช้งาน Linux Terminal อย่างมีประสิทธิภาพ:** ไม่เพียงแค่จำคำสั่ง แต่เข้าใจ Option และพฤติกรรมของคำสั่งอย่างลึกซึ้งในสถานการณ์ที่แตกต่างกัน
* **คิดแบบนักสืบและวิเคราะห์โครงสร้าง:** พัฒนาทักษะการสืบค้นข้อมูล การตรวจสอบประเภทของไฟล์ (File Types) และเข้าใจเรื่องการจัดการสิทธิ์เข้าถึงไฟล์ (Permissions) อย่างถูกต้อง
* **รากฐานด้าน Cyber Security:** เข้าใจภาพรวมของระบบเครือข่ายเบื้องต้น และกลไกที่มักถูกใช้เป็นช่องโหว่ในระบบจริง
