# Bandit Level 3 → 4

## Objective

Retrieve the password stored in a hidden file in the `inhere` directory.

## Skills Learned

- Hidden Files Listing
- Directory Navigation

## Commands Used

cd inhere
ls -a
cat .hidden

## What I Learned

- การสลับตำแหน่งไดเรกทอรีการทำงานด้วยคำสั่ง `cd`
- การใช้คำสั่ง `ls -a` เพื่อบังคับให้ระบบแสดงผลไฟล์ที่ถูกซ่อนอยู่ (ไฟล์ที่ขึ้นต้นด้วยจุด `.`)
