# Bandit Level 1 → 2

## Objective

Retrieve the password stored in a file named `-`.

## Skills Learned

- Handling Special Filenames

## Commands Used

cat ./-

## What I Learned

- การเปิดอ่านไฟล์ที่มีชื่อเป็นเครื่องหมายพิเศษอย่าง `-` โดยจำเป็นต้องใส่สัญลักษณ์ระบุพาธ เช่น `./-` เพื่อป้องกันไม่ให้ระบบสับสนกับตัวเลือก (Options) ของคำสั่ง
